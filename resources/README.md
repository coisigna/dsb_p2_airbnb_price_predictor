# Nomenclature

For the development of this code, we're going to use the following variables structure:

```python
var1 = "var_name"
varN = "var_name"

#list

l_{var1}_{varN}

#nparray

nparr_{var1}_{varN}

#dataframe

df_{var1}_{varN}

#dictionary

d_{var1}_{varN}

#series

ser_{var1}_{varN}

```
