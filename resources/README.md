# Nomenclatura

Para el desarrollo del código vamos a seguir la siguiente estructura para las variables:

```python

var1 = "nombrevariable"
varN = "nombrevariable"

#lista

l_{var1}_{varN}

#nparray

nparr_{var1}_{varN}

#dataframe

df_{var1}_{varN}

#diccionario

dct_{var1}_{varN}

#series

ser_{var1}_{varN}

```
